二、立项依据（项目意义、现状分析）
1.项目意义
1.1 手诊在医疗诊断中的重要性与实用性
　　手诊是指通过人体手的纹路形态、变化、规律等方式，对人体器官的演变做出推理的一种辅助防治手段。其原理是通过对手上的征象进行有目的地观察，了解人体健康或疾病状况。无论西医的视、触、叩、听，还是中医的望、闻、问、切，其视、望均排第一位；手诊主要是指对手部的望诊，这种方法中西方都有，主要分为气色形态、手纹和手形三大类。
　　传统中医认为手是人体重要的器官之一。手部皮肤的结构特点为含有丰富的血管与神经末梢，手掌表皮厚而致密，有较好的透明性，可观察到皮下气血的盈虚盛衰，故成为观察人体健康的窗口。过去的病变留下来的痕迹，现在病变的反应，以及潜伏的疾病的预兆，都会在手上呈现出来。
　　根据我们与公司合作获取的数据显示，手诊对于实际疾病识别的准确率较高，且可操作性强，无论是作为日常养生保健的参考依据还是作为现代医学的辅助参考，都有极大的实际价值。
1.2 手诊标准统一化，突破中医局限性
　　目前手掌诊病的知识只掌握在少数中医专家手里，这是因为中医需要较大的知识沉淀，中医专家在我国也是比较稀缺的，这大大限制了手掌诊病乃至中医的广泛应用。
　　通过传统中医判断依据和深度学习相结合，能够改变以往以经验感觉为主的传统诊断模式，实现手诊判断标准统一化、自动化。在通过对大量已有正确标注的手掌病理数据进行分析后，我们的算法将能够更精准的提取有关特征信息，对需要进行病症分析的图像进行判断分类识别，不再需要传统人工比对，这突破了中医专家稀缺的局限性，看掌诊病软件的开发也使中医能够惠及更多人，让更多人享受到中医带来的疾病预警作用。
1.3 由门诊到个人的程序推广
	 以通过大量训练后得到的深度学习模型作为基础开发得到相应的程序，可以推广运用到临床问诊上面，无论是作为检测标准还是参考依据都有助于提高对于病症的识别效率与准确率。同时也可以作为装载在移动平台或者是在线网站的程序，只要用户按照一定要求上传手掌照片，即可以通过系统自动识别相应病症用作日常参考。
1.4 促进传统中医的现代化发展
	手诊自动化在一定程度上解决了传统中医缺乏检测标准的问题，通过深度学习的识别能力，将传统中医与现代科技创新结合，有利于让普通民众再次认识到传统中医的实际作用，推动中医在社会的普及，加深普通人对中医文化的了解，以促进中华传统文化的发展。
1.5 图像识别技术与深度学习技术助力“人工智能+医疗”
　　自2012年深度学习技术被引入图像识别数据集之后，其识别率近年来屡创新高，将深度学习技术与医疗图像识别技术结合已经成为人工智能领域势不可挡的发展趋势了。
	目前最新的图像识别技术准确率已经可以超过人眼了，通过对大量图像数据的分析，深度学习技术可以更精准的提取特征点进行高精度的识别，而这一点正是手诊技术实现的局限。通过运用深度学习技术分析识别手掌图像，不但可以突破这个局限，还可以使手诊标准化统一化，实现人人即时手诊。
2.现状分析
2.1现有手掌诊病技术方案与问题
　　目前，手掌诊病技术已经有了一定的发展，但是主要停留在理论层面。其中应用最广的是掌纹线特征提取技术。掌纹线特征提取技术是掌纹识别系统和掌纹自动诊病系统的关键部分，其目的是为系统提供尽可能多的、可处理的有效信息，处理结果的优劣将直接对决策系统产生干扰。掌纹线特征属于掌纹的结构特征，这种基于物理结构的特征最为直观，也最符合掌纹诊病的提取要求[1]。
　　现有的识别手掌异常纹的算法有基于掌纹无向图的模糊识别法、模板匹配法、领域编码法、霍夫变换法和几何特征法。①基于掌纹无向图的模糊识别法[2]。廖静把掌纹的端点、交点以及连结信息用无向图表示出来，并结合模糊数学的方法进行识别异常纹，虽然无向图表示方法可以快速提取出掌纹线的大致信息，但是无向图是对真实纹线的一种近似表达，忽略了端点之间的信息。此外，还需要根据不同情况手动调整隶属度参数；②模板匹配法[3]：此方法需要大量的预处理工作，这对识别的精度干扰很大；③领域编码法[4]：根据纹线形状进行编码，对于“十”字纹和“米”字纹的识别效果不错，但是此方法难以推广到其他封闭型异常纹的识别；④霍夫变换法[5]。应用霍夫变换的累加器投票原理对“十”字纹和“米”字纹来进行识别，识别效率有所提高，但此方法仅局限于开放型异常纹的识别，同样不适合封闭型纹的识别；⑤几何特征法。文献[6]抓住不同形状的物体的面积和周长之比不一样的特点，并结合物体轮廓边界点到其中心的距离变化的倒数关系对异常纹来进行识别，对一些异常纹的识别取得一些效果，但是各个纹路之间的识别依据存在一定的重合，难以进一步准确区分。比如，“＃”字纹无法准确识别，某个满足方行纹的识别特征同时也满足星状纹的判断依据。
　　现有的掌纹颜色提取所运用的颜色空间模型有RGB颜色模型、HIS颜色空间模型和HSV颜色模型。①RGB颜色模型：在自然界中任何一种颜色都可以由R、G、B三基色按不同的比例混合而组成。RGB色彩空间采用物理三基色表示，因而物理意义很清楚，适合彩色显象管工作。这个标准几乎包括了人类视觉所能感知的所有颜色，是目前运用最广的颜色系统之一，但是RGB颜色空间并不与人的视觉特点相吻合[6]。②HIS颜色空间模型：HIS颜色空间模型[6]可以降低彩色图像处理的复杂性，加快数字图像处理的速度。其优点是可以非常直观地描述彩色，与HSV相近。③HSV颜色模型：HSV（Hue，Saturation，Value）颜色模型是根据颜色的直观特性由A.R.Smith在1978年创建的一种颜色空间，通常也被称为六角锥体模型（Hexcome Model）。HSV颜色模型类似于人类感觉颜色的方式。HSV以人类更熟悉的方式封装了关于颜色的信息：“这是什么颜色（H）？深浅如何（S）？明暗如何（V）？”。为使彩色图像处理过程与手掌专家诊病原理更为接近[7]。本项目将会使用HSV颜色模型来进行手掌彩色图像预处理。
浏览已有的几种解决方案，发现其存在的问题主要有如下三点：
（1）现有方法提取的掌纹识别主线浅且易断续，不利于后续处理。
（2）现有方法极少可用于提取一些异常纹路。
（3）手掌颜色能够反映整个身体健康状况，现有方法不能识别手掌上的颜色斑点等。
　　综上，看掌诊病受传统方法局限，仅停留在理论阶段，运用深度学习能够很好地解决这个问题，能够推动看掌诊病从理论到实践的发展。
2.2国内外政策推广传统中医发展

　　不单是在国内，近几年，中医在美国可谓日益风行，并越来越受到重视。去年，美国的一份调查报告显示，看过中医的人、愿意接受中医治疗的人已经超过50%。全美一年每人平均接受中医服务的次数近2次。在美国从事中医药相关工作的人差不多有4.5万人。2002年，美国白宫还批准将世界上43种传统医学和疗法正式纳入美国补充和替代医疗体系，其中“中国传统医学”作为独立医学体系正式被列入白宫文件。美国政府每年花费超1.2亿美元用于补充和替代医学的研究和发展，而针对中医、中药和针灸的研究项目多达几十种。
	 中国、美国、韩国、日本……国内外都对传统中医愈加重视，而手诊作为中医诊断的中医一部分，也急需得到科学的发展。

三、项目实施方案、实施计划及可行性分析（按项目内容、目标、
拟解决的关键问题、实施方案、具体实施计划和可行性分别详细填写）
1. 项目内容
（1）基于单次多盒检测器的手掌区域检测
（2）基于级联深度卷积神经网络的手掌特征点定位
（3）基于密集连接卷积网络的症状识别
（4）手掌图像标注软件的开发
2. 项目目标
　　本项目基于深度学习和中医理论基础，通过对与企业合作获取得到的中医专家标注病理的手掌图像数据，并通过手掌区域检测、关键点定位、症状匹配实现看掌诊病算法的研发以及看掌诊病App的开发。
3. 拟解决的关键问题
（1）如何准确判断图片中手掌的有无
（2）如何精确定位手掌在图片中的位置
（3）如何通过手掌掌纹识别准确判断出身体状况
4. 实施方案
（1）基于单次多盒检测器的手掌区域检测
　　单次多核检测器基于前馈卷积网络，该网络对输入的手掌图像进行卷积，产生固定大小的边界框集合，并对这些边界框中存在的手掌类别实例进行评分，然后进行非极大值抑制步骤来产生最终的检测结果，判断手掌有无。
（2）基于级联深度卷积神经网络的手掌特征点定位
　　我们通过人工设计操作（如卷积、局部或全局编码方法）将输入数据转换成合适的形式。同时还要根据手掌特征手动设计一些不变量。通过这种转换，我们能够将输入数据表征成一种更容易分离或识别的形式，通过特征点的定位我们可以获取掌心的具体位置，从而有利于病状匹配的进行。
（3）基于密集连接卷积网络的症状识别
　　密集连接卷积网络通过特征复用来发掘网络的潜能，生成易于训练的和参数高效化的精简模型。连接不同层学习的手掌特征图可以增加后续层的输入的变化并提高效率。我们通过使用密集连接卷积网络对掌心区域进行识别从而获取病状结果。
（4）手掌图像标注软件的开发
　　深度学习以大量数据为先导，本项目中手掌区域定位以及特征点识别所需的ground-truth均需要人为标注，我们需要开发一个标注软件来获取手掌区域与手掌关键点的坐标位置。
5. 具体实施计划
　　我们通过与企业合作，目前已经获取中医专家标注的标注病理的手掌图像25000幅，总共分为45类病状。本项目拟研发基于深度学习的中医手掌诊病系统，项目拟采用的技术路线如图1所示。下面还对每个核心技术模块的研发思路进行详细阐述。

（2）基于级联深度卷积神经网络的手掌特征点定位
2.1一级特征点预估

图3 卷积实现
　　深度卷积神经网络(Deep Convolutional Neural Network,DCNN)包括感受野、通道、滤波器组、特征图、激活函数、卷积层和池化层等。图3所示卷积层感受野尺寸为r×s，不同的滤波器沿着空间维度在输入层上滑动，生成多个特征图，图3所示卷积操作后的特征图空间尺寸为(m-r+1)×(n-s+1)，通道数由卷积层的滤波器的个数决定，滤波器的总和称为滤波器组。如图3所示，通过一个卷积层后，第t层第k个特征图在空间位置(i,j)处的元素由公式(4) 计算：
    〖  a〗_(i,j,k)^((t))=f(∑_(c=1)^p?∑_(m=1)^s?∑_(n=1)^r?a_(i+m,j+n,c)^((t-1) ) ×w_(m,n,c,k)^((t) )+b_k^((t) ) )        (4)            
　　其中，f(?)为逐元素运算的激活函数，s，r是感受野空间尺寸，p为输入层通道数，b为偏置项。a^((t-1))，a^((t))分别为t-1层和t层的响应，w为权值。则卷积层一个空间位置的响应值的矢量化形式为：
〖                                           a〗^((t))=f(W^((t) ) a^((t-1) )+b)                             （5）
　　其中a^((t-1))∈R^(s×r×p)，W∈R^(q×(s×r×p))，其每行是一个矢量化的滤波器, b^((t))∈R^q。对第t层输出的第k个特征图进行池化操作如公式(5)所示。
                           a_(i,j,k)^((t+1))=(max)┬(1≤m≤u,1≤n≤v) a_(i×u+m,j×v+n,k)^((t) )                    （6）                        
　　其中u×v大小的区域称为池化邻域。
    全连接层的计算方式如式（7）所示。
　　                  a_i^((t+1))=f(∑_(j=1)^p?〖a_j^((t))×w_j^((t+1))+b^((t+1)))               （7）〗                      
　　用矢量化表示第t+1层的输出，则可表示为：
　　  〖                                    a〗^((t+1))=f(W^((t+1) ) a^((t) )+b^((t) ) )                        （8）
　　其中，a^((t))是第t层的输出，假定其维数为p，则p为t层的神经元个数。b^((t))为偏置矢量。全连接层的输出是简单的一组线性排列的神经元，假定其维数，即神经元的个数为q，则参数是q×p的矩阵。
    忽略池化及其它不包含可学习参数的连接层，假定a^((1))=x表示输入层，而a^((t))表示第t层的响应，基础的深度卷积神经网络可以表示为式(9)：
z^((t+1))=W^((t)) a^((t))+b^((t))
　　〖                                            a〗^((t+1))=f^((t+1) ) (z^((t+1) ) )                                （9）
    其中，z，a和b是响应和偏置的矢量表示，W是权值的矩阵形式，f是逐元素的激活函数。
    假定整个网络模型用h表示，T表示可学习层的数目，所有权值和偏差的集合用{W,b}表示，则由式(9)模型h可表示为式(10)的形式。
   h{W,b}=f^((T) ) (W^((T) ) (...f^((2) ) (W^((2) ) f^((1) ) (W^((1) ) a^((1) )+b^((1) ) )+b^((2) ) )+b^((T) ) )     (10)
    我们首先利用一个9层的深度卷积神经网络实现手掌特征点大致区域定位。由于网络的最后两层是全连接层，输入图像的尺寸必须是固定大小，因此针对候选的手掌区域，将其缩放到39×39像素大小。对于输入图像，先后用一个4×4和两个3×3 的卷积核对图像进行卷积操作，每个卷积层后面进行最大值池化来降低参数的数量,提高网络的效率。最后，通过两个全连接层对坐标参数进行回归,由于实验中采用了6个特征点的标注，最终输出层的神经元数目为12,代表了回归出的6个特征点的二维坐标值。
2.2次级特征点优化网络
　　LPRCRF在每个回归阶段包含两个步骤,即局部特征学习和全局回归。基准点的坐标，记做S=[x1,y1,...,xn,yn]^T，其中n是基准点的个数。在特征学习阶段，特征提取器通过式(11)学习。
                                                (min)┬(φ_i^t ) γ(φ_i^t (I,S_i^(t-1) ))                                        (11)
     其中I是手掌图像，S_i^t"-1" 是(t-1)^th层i^th基准点的坐标预测值，φ_i^t是在t^th阶段针对i^th基准点的特征提取器，γ(?)是由特定特征学习算法确定的损失函数。训练时，通过优化特征提取器的参数使损失函数值最小。
    此后，各个特征提取器串联形成一个全局特征提取器Φ^t=[φ_1^t,φ_2^t,...,φ_n^t]。而后会学习一个全局特征提取器Φ^t和形状残差ΔS^t=S┴^-S^(t-1)之间的全局回归器，其中的形状残差是形状真值S┴^和预测形状S^(t-1)之间的差值。通常通过式(12) 所示的L2正则化回归来学习该映射。实际中应用Liblinear 库来进行回归。
　　　                              (min)┬(W^t ) ∑?〖|(|ΔS^t-W^t Φ^t | |_2^2+λ)||W^t | |_2^2 〗                    (12)
　　其中，W^t是回归参数，λ是正则化系数。
　　而后基于学习到的局部特征和全局回归应用式(13)进行形状更新。
　　　　Δ (S^t)┴~=W^t Φ^t (I,S^(t-1))
                                                   S^t=S^(t-1)+(S^t)┴~                                      （13）                         
　　通过一级的神经网络，我们得到了手掌特征点的大致位置。我们通过六个浅层的神经网络组成次级特征点优化网络，针对特征点所在的局部区域进行分析和处理，从而回归出特征点的精确位置。
　　
　　图 4 手掌特征点定位
　　六个子网络分别针对图4中六个特征点进行坐标的回归。选取一级网络得到的特征点周围15×15的区域，分别作为六个子网络的输入图像。我们先后采用一个4×4和一个3×3的卷积核对局部图像进行卷积操作，每个卷积层后面接最大值池化层操作。最后是两个全连接层，由于每个子网络只针对一个点进行回归，因此输出层的神经元数目为2,代表了对应坐标点的二维坐标值。由于次级网络的坐标回归是在局部图像上进行的，对于次级网络的输出需要进行线性坐标映射还原到原图上去，从而形成最终的网络输出。

（3）基于密集连接卷积网络的症状识别
　　考虑一个通过卷积网络的单个图片x_0，该网络包含L层，每个层实现一个非线性转换H_l (・)，其中l表示层的编号，H_l (・)可以是操作的复合函数，比如Batch Normalization(BN)[5]、整流线性单元(ReLU)、池化层和卷积层。我们将第l层的输出定义为x_l。
　　传统的前馈卷积网络连接第l层的输出作为第l+1层的输入，从而产生了下面的层转换:
　　〖                                                    x〗_l=H_l (x_l-1)                                         (14)
　　密集连接卷积网络（Densely Connected Convolutional Networks，DenseNet）采用的连接模式：从任何层向其所有后续层的直连接，如图6所示。因此，第l层接受其前所有层的特征图作为输入：
　　
　　图 6  DenseNet网络框架
                                          〖  x〗_l=H_l ([x_0,x_1,…,x_(l-1) ])                              (15)
　　其中[x_0,x_1,…,x_(l-1) ]表示第0到l-1层产生的特征图的连接。为了便于应用，我们将上式中的多输入[x_0,x_1,…,x_(l-1) ]连接成一个张量。
　　将H_l (・)定义为一个复合函数[6]，它由三部分连续的操作组成：Batch Normalization、ReLU和一个3×3的卷积操作。
　　为了便于在我们的体系结构中使用下采样层，我们将网络划分为多个紧密相连的密集块，如图7所示。我们将块之间的层称为过渡层，它们进行卷积和池化操作。我们实验中使用的过渡层包括一个BN层和一个1×1的卷积层，其次是一个2×2的平均池化层。
